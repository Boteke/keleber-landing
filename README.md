# Keleber — публичный сайт

Лендинг и юридические страницы сервиса записи **Keleber** (запись клиентов через WhatsApp, Казахстан).

## Структура

- `index.html` — лендинг
- `privacy.html` — политика конфиденциальности
- `terms.html` — условия использования

## Перед публикацией (для подачи в Meta)

1. Заполнить заглушки — поиск по слову `ЗАГЛУШКА`: название юрлица, `[EMAIL]`, `[ТЕЛЕФОН]`, `[АДРЕС]`, `[ДАТА]`, год.
2. Юридические тексты (`privacy.html`, `terms.html`) — рабочий **шаблон**, не юридическая консультация. Перед публикацией показать юристу, особенно по части трансграничной передачи данных через Meta (закон РК «О персональных данных и их защите»).
3. Проверить, что `privacy.html` и `terms.html` открываются по ссылкам из футера.

## Языки

RU / KZ — переключатель в шапке каждой страницы.

## Деплой

Статические HTML-файлы. Ссылки между страницами относительные (`index.html`, `privacy.html`, `terms.html`) — работают на любом хостинге, включая GitHub Pages.

**Сайт:** https://boteke.github.io/keleber-landing/

Кастомный домен `keleber.kz` подключать только после настройки DNS. До этого держите поле Custom domain в GitHub Pages пустым и не добавляйте `CNAME` в репозиторий.
